■jinja2とvuejsを一緒に使う時、出力方法が同じのためコンフリクトが起こる問題
解決：
どちらかのデリミタを変更する。
〇jinja2の場合
from flask import Flask

class CustomFlask(Flask):
  jinja_options = Flask.jinja_options.copy()
  jinja_options.update(dict(
    block_start_string='(%',
    block_end_string='%)',
    variable_start_string='((',
    variable_end_string='))',
    comment_start_string='(#',
    comment_end_string='#)',
  ))

app = CustomFlask(__name__)

→{{}} を (()) に変更する。

〇vuejsの場合
var app = new Vue({
  el: "#app",
  delimiters: ["[[", "]]"],
  data: {
    message: "Hello Vue!"
  }
})
→{{}} を [[]] に変更する。
